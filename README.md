# Pharmaceutical Sales & Drug Demand Forecasting Dashboard

## Project Overview
Power BI dashboard analyzing 2,000+ pharma sales transactions across India (2023-2024).

## Dashboard Pages
1. Sales Overview - KPIs, trends, top drugs
2. Regional Analysis - India map, distributor performance
3. Inventory & Expiry Risk - Stock health, expiry alerts
4. Demand Forecasting - 12-month forecast

## Tools Used
- Power BI Desktop
- DAX (15+ measures)
- Excel Dataset

## Dataset
- 2,000 sales transactions
- 30 drugs across 6 categories
- 5 regions, 10 warehouses, 10 distributors

## Key Insights
- Oncology drugs drive highest revenue (63%)
- Rituximab is top selling drug
- South India highest revenue region
